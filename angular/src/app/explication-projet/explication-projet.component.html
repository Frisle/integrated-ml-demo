The Titanic Dataset

The Titanic Dataset is a dataset composed the data of 891 passengers that were on the Titanic when it sank. For each passenger, we have part or all of the following informations :

  • name: the name of the passenger
  • sex: the sex of the passenger
  • age: the age of the passenger
  • survived: the passenger survived or not
  • class: the class the passenger traveled in
